Getting Started With ConveyAPI - Part 1.

Tweet

Revealed Context provides a proven mix of text analytics and machine learning techniques so that your data can be analyzed with the highest accuracy. Machine learning approaches allow computers to detect patterns from large amounts of data. You can use human analysts to identify by hand the relevant records with the word “Coke” vs. the records that are discussing drugs or steel processing. You could even segregate the records that truly are discussing the brand vs. those that say, “5 pm—ready for my rum and coke!” Once you accumulate enough human-coded training data, the machine learning technology can discern the patterns that separate the relevant from the irrelevant. Similarly, machine learning can more accurately separate the positive from the negative sentiment.

That’s what Revealed Context technology can do.

Revealed Context offers two different technologies, both of which use state-of-the-art text analytics and machine learning techniques to analyze social media data:

  • Concur. Train your relevancy model to quickly create a custom classifier to separate relevant vs. irrelevant data.
  • Convey. Use our models trained with hundreds of thousands of coded social media conversations to detect sentiment, emotion, and more.

Today, I’ll explain just how to use general sentiment analysis with ConveyAPI which is already trained by tons of training data. So, you can start analyzing your data right away without training anything. Of course, it won’t be optimized for your specific domain, but it will do general sentment analysis, such as polarity, emotion, intensity, and spam.

First, you need a API key to get authenticated for REST API service use. For now, just email me (rbaek@revealedcontext.com) and I’ll create free testing account for you. Remember! Don’t share any of your API keys with anyone.

Once you obtain the key, you’re all set!

If you want to analyze your text,

“the service wasn’t the best but the food more than made up for it”

Create URL call,

  • Method URL: http://beta.conveyapi.com/analysis-engine/process
  • Method Actions: GET, POST
  • Parameters: api_key, text, annotation.emotion

Your final URL will be like this.

http://beta.conveyapi.com/analysis-engine/process?api_key=YourAPIKey&text=the+service+wasn%27t+the+best+but+the+food+more+than+made+up+for+it&annotation.emotion=true

And this will return response in Json format like below.

You can also send us request in batch mode, just using Method URL http://beta.conveyapi.com/analysis-engine/process-batch Note that batch array must be indexed starting with 0, and moving up sequentially. For example,

curl -v -X POST "http://ubervu.conveyapi.com/analysis-engine/process" -d "api_key=YourAPIKey&annotation.level=sentence&annotation.polarity=true&annotation.emotion=true&annotation.intensity=true&annotation.spam=true&text=In+the+Grand+Canyon%2C+Arizona+has+a+natural+wonder+which%2C+so+far+as+I+know%2C+is+in+kind+absolutely+unparalleled+throughout+the+rest+of+the+world.+I+want+to+ask+you+to+do+one+thing+in+connection+with+it+in+your+own+interest+and+in+the+interest+of+the+country+-+to+keep+this+great+wonder+of+nature+as+it+now+is."

For more details on parameters, please refer to active docs

What’s next? Create your own classifier using your own dataset with concour! We’re currently working on enhancing this application, there will be a lot of improvements coming soon. It will be covered in the next blog post.
